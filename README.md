## Install

```
git clone git@github.com:anwert-io/anwert-mjml-boilerplate.git <Your Project>
cd <Your Project>
npm install
```

## Usage

### Unix-based

**Building**
```
npm run build
```

**Watching**
```
npm run watch
```

### Windows

**Building**
```
npm run build-windows
```

**Watching**
```
npm run watch-windows
```
